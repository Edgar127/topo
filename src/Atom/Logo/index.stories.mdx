# Logo

This component renders logos for our programs, with our without text, and in various colors. It takes the following
props:

- `withText` -- If this is set, the logo includes the full lockup with the program name.
- `textColor` -- The color or color ID of the text. Defaults to the page text color.
- `logoColor` -- The color or color ID of the logo. Defaults to the current theme brand color.
- `text` -- Override the text displayed with the logo.

The logo component composes Box. You can pass any props you'd usually pass to a `<Box />`, except that it will not
respect `color`, and that the default display is `inline` (however this can be overridden as usual).

## Custom Text

You can also set `text` to override the text shown. Note that the text will render in unkerned Sofia Pro, while our
actual logos are in tightly-kerned Proxima Nova, so it won't really look right and should mainly be used for microsites.

Here's an example with custom color and text:

    </>
