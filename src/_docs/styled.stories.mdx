# Styled System

All components can be configured using [styled-system](https://styled-system.com/) props.

## Responsive Design

All styled-system props can respond to viewport widths by setting its value to a width:

```jsx
  <Grid
    templateColumns={{
      base: "1fr",
      sm: "repeat(1fr, 2)",
      md: "repeat(1fr, 3)",
      lg: "repeat(1fr, 4)",
      xl: "repeat(1fr, 6)",
    }}
  />
```

You may omit breakpoints, in which case the breakpoint will be configured by the next specified breakpoint.

(The exact breakpoint widths are set in the `breakpoints` theme var.)
