# Header

This represents a standard header for full websites. It takes the following props:

- `darkBackground` Inverts the gradient, outline, and -- if possible -- text. See below for details.
- `underscore`: Adds an underline to the menu. This defaults off but should _usually_ be on.
- `gradAmount` (`null`, `"sm"`, or `"lg"`): Adds a weak or strong gradient behind the header to ensure text is readable
  with an image background. Defaults to `sm`.

## Usage

```jsx
import Header, { SiteLogo, Menu } from '@codeday/topo/Header';
import { CodeDay } from '@codeday/topo/Logo';
import Button from '@codeday/topo/Button';
import Link from '@codeday/topo/Link';

export default () => (
  <MyPage>
    <Header underscore>
      <SiteLogo>
        <CodeDay withText />
      </SiteLogo>
      <Menu>
        <Link href="/about">About</Link>
        <Button href="/register">Register</Button>
      </Menu>
    </Header>
  </MyPage>
)
```

## Dark Background

If you set `darkBackground`, the header will attempt to recolor itself to support dark backgrounds. It can automatically
re-color only the following Topo components:

- `<Logo />`
- `<Link />`
- `<Button />` (but it will not do so if a `variantColor` is set)

_This is separate from Topo's regular support for dark mode, and is intended solely for use when something is guaranteed
to be a dark background, such as when displaying the menu on top of a hero cover image._
