# Installation

```
yarn add @codeday/topo
```

```
npm add @codeday/topo
```

## Other dependencies

If you are using `@chakra-ui/react` please use version `~1.6.12`

```
yarn add @chakra-ui/react@~1.6.12
```

```
npm add @chakra-ui/react@~1.6.12
```
