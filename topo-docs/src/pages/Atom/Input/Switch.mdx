import { Box, Switch, FormControl, FormLabel, Stack } from "@codeday/topo/Atom";
import { Playground } from "@nexite/dokz";

# Switch

The `Switch` component is alternative to the
[Checkbox](/Atom/Input/Checkbox) component. You can switch between enabled or
disabled states.

## Import

```js
import { Switch } from "@codeday/topo/Atom";
```

## Example

<Playground>
  <Box p={3}>
    <FormControl display="flex" alignItems="center">
      <FormLabel htmlFor="email-alerts" mb="0">
        Enable email alerts?
      </FormLabel>
      <Switch id="email-alerts" />
    </FormControl>
  </Box>
</Playground>

## Sizes

Pass the `size` prop to change the size of the `Switch`.

<Playground>
  <Box p={3}>
    <Stack align="center" direction="row">
      <Switch size="sm" />
      <Switch size="md" />
      <Switch size="lg" />
    </Stack>
  </Box>
</Playground>

## Switch background color

You can change the checked background color of the `Switch` by passing the
`colorScheme` prop.

<Playground>
  <Box p={3}>
    <Stack direction="row">
      <Switch colorScheme="red" />
      <Switch colorScheme="teal" size="lg" />
    </Stack>
  </Box>
</Playground>
