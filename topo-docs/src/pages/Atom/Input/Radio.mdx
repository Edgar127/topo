# Radio

Your typical `Radio` button.

## Import

```js
import { Radio } from "@codeday/topo/Atom";
```

## Example

<Playground>
  <Box p={3}>
    <RadioGroup defaultValue="1">
      <Stack direction="row">
        <Radio value="1">First</Radio>
        <Radio value="2">Second</Radio>
        <Radio value="3">Third</Radio>
      </Stack>
    </RadioGroup>
  </Box>
</Playground>

## Radio with custom color

You can override the `colorScheme` of the Radio to any color key specified in `theme.colors`.

<Playground>
  <Box p={3}>
    <RadioGroup defaultValue="2">
      <Stack spacing={5} direction="row">
        <Radio colorScheme="red" value="1">
          Radio
        </Radio>
        <Radio colorScheme="green" value="2">
          Radio
        </Radio>
      </Stack>
    </RadioGroup>
  </Box>
</Playground>

## Note about name prop

Pass the `name` prop to the `RadioGroup` component rather than the `Radio` component. By default, the `name` prop of the RadioGroup takes precedence.

```js
// Do this ✅
<RadioGroup name="form-name">
  <Radio>Radio 1</Radio>
  <Radio>Radio 2</Radio>
</RadioGroup>

// Don't do this ❌
<RadioGroup >
  <Radio name="form-name">Radio 1</Radio>
  <Radio name="form-name">Radio 2</Radio>
</RadioGroup>
```
