# Button

When you want something to be clickable, use a button!

## Import

```js
import { Button } from "@codeday/topo/Atom";
```

## Props

- `variant` -- One of `"solid"` (default), `"outline"`, or `"ghost"`
- `colorScheme` -- Any brand color, without the shade (e.g. `green`, but not `green.500`). Default is gray.
- `size` -- One of `"xs"`, `"sm"`, `"md"` (default), `"lg"`.
