# CodeBlock

You can render blocks of code with the `<CodeBlock />` tag:

## Import

```js
import { CodeBlock } from "@codeday/topo/Atom";
```

## Example

<Playground iframe>
  <CodeBlock lang="python" numbers={true}>
    {`@requires_authorization
def somefunc(param1='', param2=0):
  """This method does the thing"""
  if param1 > param2: # interesting
    print 'Gre\\'ater'
  return (param2 - param1 + 1 + 0b10l) or None
somefunc('foo')`}
  </CodeBlock>
</Playground>

It takes two params:

- `numbers`: Whether to show line numbers or not.
- `lang`: The language to highlight in.
