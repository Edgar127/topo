# CodeDay Logos

Topo has components for many of the CodeDay brand logos. Use the `withText` prop to display the text.

<Playground>
  <Box p={3} fontSize="4xl">
    <VStack align="left">
      <CodeDay withText />
      <CodeCup withText />
      <Labs withText />
      <CsFest />
      <Evangelist />
      <Clear />
      <Community />
    </VStack>
  </Box>
</Playground>

## Import

```js
import {
  CodeDay,
  CodeCup,
  Labs,
  CsFest,
  Evangelist,
  Clear,
  Community,
} from "@codeday/topo/Atom";
```

## Logo with custom withText

You can use the `text` prop to set custom text.

<Playground>
  <Box p={3} fontSize="4xl" align="left">
    <CodeDay text="Topo is cool!" />
  </Box>
</Playground>

## Changing the size of the logo

You can use the `fontSize` prop change the size of the logo.

<Playground>
  <Box p={3}>
    <VStack align="left">
      <CodeDay fontSize="4xl" text="4xl" />
      <CodeDay fontSize="3xl" text="3xl" />
      <CodeDay fontSize="md" text="md" />
    </VStack>
  </Box>
</Playground>
