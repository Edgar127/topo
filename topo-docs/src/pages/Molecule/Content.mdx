# Content

Most of your content should be in a `Content` box, which sets a standard width and `margin: 0 auto`. Y(ou can set the
`wide` or `full` props to make the content wider.)

<Playground>
  <>
    <Header underscore>
      <SiteLogo>
        <Clear withText />
      </SiteLogo>
      <Menu>
        <Button href="#">Hi</Button>
      </Menu>
    </Header>
    <Content wide>This is a wider box, for emphasis.</Content>
  </>
</Playground>

## Import

```js
import { Content } from "@codeday/topo/Molecule";
```
